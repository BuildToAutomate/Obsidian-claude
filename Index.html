⚡ 12 commands that turn notes into ideas

📄 Surface patterns across years 💡 Generate grounded ideas 🔄 Track belief evolution
Each command includes what it does, when to use it, and a ready-to-adapt prompt. Turn Obsidian into a true thinking partner.
command 01
Unlinked Mention Harvest
🔗 latent connections
📌 WHAT IT DOES
Finds notes that mention the current note's title but aren't linked yet. Surfaces hidden, implicit relationships.
⏱ WHEN TO USE
When you suspect a note is more relevant than backlinks suggest.
✨ EXAMPLE PROMPT (Dataview)
LIST FROM "" WHERE contains(file.content, this.file.name) AND !contains(file.outlinks, this.file.name)
command 02
Cluster by Co‑occurring Tags
🏷️ emergent themes
📌 WHAT IT DOES
Groups notes sharing 2+ uncommon tags, revealing hidden thematic clusters across years.
⏱ WHEN TO USE
Discover emergent topics or unexpected idea constellations.
✨ EXAMPLE PROMPT (Dataview)
TABLE tags, file.ctime FROM #insight OR #question OR #contrarian WHERE length(filter(tags, (t) => !contains(["daily","reference"], t))) > 1 SORT file.ctime desc
command 03
Evolution of a Belief
⏳ track shifts
📌 WHAT IT DOES
Extracts all notes containing a specific opinion/phrase, sorted chronologically to see how thinking changed.
⏱ WHEN TO USE
Review personal intellectual history or a concept transformation.
✨ EXAMPLE PROMPT (search)
"believe that" OR "I used to think" OR "my stance on X" → sort by file.ctime
command 04
Abandoned Sparks
🔥 forgotten seeds
📌 WHAT IT DOES
Lists notes with high potential signals (“maybe”, “interesting”) untouched for 6+ months.
⏱ WHEN TO USE
Rediscover dormant ideas before they rot.
✨ EXAMPLE PROMPT (Dataview)
LIST FROM "" WHERE contains(file.content, "maybe") OR contains(file.content, "what if") AND file.mtime < date(today) - dur(6 months) SORT file.mtime asc
command 05
Cross‑Year Juxtaposition
⚖️ creative tension
📌 WHAT IT DOES
Pairs two notes from different years sharing a core concept but contradicting each other.
⏱ WHEN TO USE
Generate insights from dialectical disagreement across time.
✨ EXAMPLE PROMPT (manual + semantic)
Search "concept: productivity" → pick note 2021 vs 2025 → ask: "What would past me argue with current me?"
command 06
Recurring Question Detector
❓ thought loops
📌 WHAT IT DOES
Finds the same/similar questions asked across different contexts and years.
⏱ WHEN TO USE
Spot loops in your thinking and unresolved tensions.
✨ EXAMPLE PROMPT (regex + Omnisearch)
^.*\?.*$ → then manually review semantic duplicates: "How to prioritize?" vs "What should I work on next?"
command 07
Map of Contradictions
🗺️ dialectical pairs
📌 WHAT IT DOES
Displays linked notes with low tag overlap but connected via common third notes.
⏱ WHEN TO USE
Find opposing idea pairs that fuel new synthesis.
✨ EXAMPLE PROMPT (Dataview)
LIST FROM [[]] WHERE !(all(flat(list(file.tags, this.file.tags)))) AND length(file.inlinks) > 2
command 08
Pattern Across N Years
📆 thematic cycles
📌 WHAT IT DOES
Pulls one line from each year’s “yearly review” or December reflection to reveal emotional or thematic cycles.
⏱ WHEN TO USE
Identify repeating patterns in mood, focus, or intentions.
✨ EXAMPLE PROMPT (Dataview table)
TABLE choice(contains(file.name, "2021"), "fear", contains(file.name, "2022"), "ambition", contains(file.name, "2023"), "burnout") as Theme FROM "Reviews"
command 09
Orphaned Insights
🧩 rich isolation
📌 WHAT IT DOES
Finds notes with high internal structure (headings, bold, bullets) but zero outgoing links.
⏱ WHEN TO USE
Capture isolated gems and integrate them into your network.
✨ EXAMPLE PROMPT (Dataview)
LIST FROM "" WHERE length(file.outlinks) = 0 AND length(file.content) > 500 SORT length(file.content) desc
command 10
Mood‑Concept Correlation
😌⚡ energy mapping
📌 WHAT IT DOES
If you tag moods (#curious, #frustrated), finds which concepts appear most often per mood state.
⏱ WHEN TO USE
Understand what topics energize vs drain you.
✨ EXAMPLE PROMPT (Dataview GROUP BY)
TABLE rows.file.link as Notes, length(rows) as Count FROM #curious OR #frustrated FLATTEN file.etags as tag WHERE tag != "#curious" AND tag != "#frustrated" GROUP BY tag SORT Count desc
command 11
Silent Evolution Detector
📝 subtle shifts
📌 WHAT IT DOES
Compares how you described the same person/idea/book 1+ years apart, using diff on two notes.
⏱ WHEN TO USE
Track belief drift without explicit “I changed my mind”.
✨ EXAMPLE PROMPT (Git diff style)
diff note_2023.md note_2025.md | grep "^[<>]" → reveals changed language, tone, conclusions
command 12
Random Triplet Idea
🎲 forced breakthrough
📌 WHAT IT DOES
Pulls 3 unrelated notes (different folders, no shared tags) and forces a combinatorial idea.
⏱ WHEN TO USE
Creative block? Constraint-based ideation for fresh perspectives.
✨ EXAMPLE PROMPT (Dataview random)
LIST FROM ("-" + join(flat(rows.file.link), " ")) WHERE file.folder != "Templates" GROUP BY random() LIMIT 3
